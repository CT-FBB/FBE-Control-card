# OLT Single Control Board Risk Dashboard

Interactive dashboard to visualize and manage OLT Single Control Board redundancy risks (Section 1 & 2), filtered to exclude lab/test environments.

## Features
- Interactive Charts (Chart.js)
- Risk Segmentation (Vulnerable vs. Lack of Redundancy)
- Real-time Data Filtering
- Lab Device Auto-Filtering

## How to view
This project is optimized for GitHub Pages. You can view the live dashboard once deployed.
